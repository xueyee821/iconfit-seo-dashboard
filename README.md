# Iconfit SEO Dashboard

每周一自动更新的 SEO 监控 dashboard，数据来源：Google Search Console + GA4。

**公开访问：** https://xueyee821.github.io/iconfit-seo-dashboard/

## 数据内容

- Google Search Console：关键词排名、点击、曝光、CTR
- Google Analytics 4：流量来源、页面表现、跳出率
- 技术检测：安全 Headers、REST API、AI 爬虫可访问性、llms.txt

## 自动更新

GitHub Actions 每周一 09:00（马来西亚时间）自动运行，无需人工操作。
